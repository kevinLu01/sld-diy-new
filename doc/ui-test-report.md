# UI 测试报告（agent-browser）

日期：2026-02-18

## 测试范围
- 前台原型：`index.html`
- 后台原型：`admin.html`

## 测试环境
- agent-browser（本地文件访问）

## 测试用例与结果

1. 方案包一键套用价格联动
- 步骤：依次点击第 2、第 3 个“ 一键套用 ”按钮
- 预期：总价随方案切换更新
- 实际：
  - 初始：`¥26,480`
  - 点击第 2 个方案后：`¥32,800`
  - 点击第 3 个方案后：`¥46,900`
- 结果：通过

2. 场景 Tab 切换
- 步骤：点击“商用中央空调”Tab
- 预期：Tab 激活态变化
- 实际：已切换，页面状态更新
- 结果：通过

3. 后台页面可访问性
- 步骤：点击底部“admin.html”链接
- 预期：进入后台管理台页面
- 实际：成功进入
- 结果：通过
